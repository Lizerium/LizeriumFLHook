### 🔐 MISCELLANEOUS COMMANDS 🔐

> ---
>
> [!IMPORTANT]
>
> - Plugin loaded: misc **(MiscellaneousCommands.dll)**
>
>   Описание

```ini
Обновляет временные метки файла персонажа и может присваивать нейтральную фракцию fc_freelancer
```

> ---

> [!NOTE]
> Функция:

```bash
UserCmd_RefreshCharacters
```

> ---
>
> [!TIP]
> Команда:

```bash
/refresh
/refresh*
```

> [!CAUTION]
> Применение:

```bash
/refresh
/refresh*
```

> [!IMPORTANT]
> Описание:

```ini
Обновляет временные метки файла персонажа для всех кораблей на учетной записи.
```

> [!NOTE]
> Функция:

```bash
UserCmd_FreelancerIFF
```

```bash
/freelancer
/freelancer*
```

> [!CAUTION]

```bash
/freelancer
/freelancer*
```

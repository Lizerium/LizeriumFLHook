# Account Path Patch для FLServer

Патч для Freelancer, позволяющий изменить путь к директории аккаунтов сервера (**FLServer**).

---

## Описание

Данный патч даёт возможность перенести папку с аккаунтами в любое удобное место, вместо стандартного расположения.

Полезно, если:

- нужно хранить данные на другом диске
- используется нестандартная структура сервера
- требуется изоляция данных

---

## Установка

1. Скопируйте файлы из папки:

```
bin
```

в директорию:

```
Freelancer/EXE/
```

2. Откройте файл:

```
dasrvcom.ini
```

3. В секции `[Libraries]` добавьте:

```
acctpath_patch.dll
```

---

## Конфигурация

Настройки находятся в файле:

```
acctpath_patch.cfg
```

Укажите в нём путь к директории с данными аккаунтов.

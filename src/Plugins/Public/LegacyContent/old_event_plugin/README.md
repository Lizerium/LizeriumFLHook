# Mining Control Plugin (v1.1)

Плагин для FLHook (версии 1.6.0), добавляющий серверный контроль над процессами добычи ресурсов (mining) в Freelancer.

---

## Описание

Этот плагин позволяет:

- централизованно управлять добычей ресурсов на сервере
- контролировать поведение игроков при майнинге
- заменить стандартную проверку `lootspawn` из античита

При этом уровень защиты остаётся сопоставимым со стандартным античитом.

---

## Особенности

- Полный контроль добычи со стороны сервера
- Замена античит-проверки `lootspawn`
- Обнаружение читов при добыче:
  - при выявлении — корабль игрока уничтожается
  - информация записывается в лог `flhook_cheaters.log`

- Поддержка бонусов для добычи:
  - по зонам
  - по оборудованию

- Гибкая настройка через `.ini`

## Установка

1. Скопируйте файл:

```
bin/minecontrol.dll
```

в директорию:

```
Freelancer/EXE/flhook_plugins/dll/
```

2. Скопируйте файл:

```
bin/minecontrol.ini
```

в директорию:

```
Freelancer/EXE/flhook_plugins/
```

---

## Конфигурация

Настройки находятся в файле:

```
minecontrol.ini
```

---

## Админ-команды

```
printminezones
```

Выводит информацию о всех стандартных зонах добычи.

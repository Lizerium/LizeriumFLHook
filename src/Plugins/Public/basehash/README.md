### BASE HASH

> [!IMPORTANT]
>
> - Plugin loaded: flhash **(flhash.dll)**
>
>   Описание:

```ini
Дает вам прозвище базы, если база уничтожена. solarhash=0
чтобы перефразировать, просто удалите FLHash.ini в папке EXE (сам файл генерируется автоматически)
Помните, что файл flhash.dll также должен находиться в папке EXE для создания списка хешей.
также убедитесь, что ваши системные INI-файлы не сжаты.
```

### 🌲 BOUNTY HUNT 🌲

> [!IMPORTANT]
>
> - Plugin loaded: bountyhunt **(bountyhunt.dll)**
>
>   Описание:

```ini
Назначает награду за убийство
```

> ---

> 1.  [Пример конфигурации `bountyhunt.ini`](../../../Binaries/bin-conf/flhook_plugins/bountyhunt.ini)

> ---

> [!NOTE]
> Функция:

```bash
UserCmd_BountyHunt
```

> [!WARNING]
> Команда:

```bash
/bountyhunt
```

> [!CAUTION]
> Применение:

```bash
/bountyhunt <charname> <credits> [<minutes>]
```

> [!IMPORTANT]
> Описание:

```ini
Предлагает награду в <credits> кредитов за убийство <charname> в течение следующих <minutes> минут
```

> ---

> Функция:

```bash
UserCmd_BountyHuntId
```

> [!WARNING]

```bash
/bountyhuntid
```

> [!CAUTION]
> Применение:


> [!IMPORTANT]

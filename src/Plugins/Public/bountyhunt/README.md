> [!NOTE]
> Function:

```bash
UserCmd_BountyHunt
```

> [!WARNING]
> Command:

```bash
/bountyhunt
```

> [!CAUTION]
> Usage:

```bash
/bountyhunt <charname> <credits> [<minutes>]
```

> [!IMPORTANT]
> Description:

```ini
Offers a bounty of <credits> credits for killing <charname> within the next <minutes> minutes
```

> [!CAUTION]

```bash
/bountyhuntid <id> <credits> [<minutes>]
```

> [!IMPORTANT]
> Description:

```ini
Offers a bounty of <credits> credits for killing the player with <id> within the next <minutes> minutes
```


> 1.  [Пример конфигурации `bountyhunt.ini`](../../../Binaries/bin-conf/flhook_plugins/bountyhunt.ini)

> ---

> [!NOTE]
> Функция:

```bash
UserCmd_BountyHunt
```

> [!WARNING]
> Команда:

```bash
/bountyhunt
```

> [!CAUTION]
> Применение:

```bash
/bountyhunt <charname> <credits> [<minutes>]
```

> [!IMPORTANT]
> Описание:

```ini
Предлагает награду в <credits> кредитов за убийство <charname> в течение следующих <minutes> минут
```

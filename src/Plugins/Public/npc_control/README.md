### ADMIN COMMANDS

> [!WARNING]
> Command:

```bash
aicreate
```

> [!CAUTION]
> Usage:

```bash
aicreate <count> <name>
```

> [!IMPORTANT]
> Description:

```ini
Creates NPCs
```

> [!WARNING]
> Command:

```bash
aidestroy
```

> [!CAUTION]
> Usage:

```bash
aidestroy
```

> [!IMPORTANT]
> Description:

```ini
Destroys all NPCs
```

> [!WARNING]
> Command:

```bash
aicancel
```

> [!CAUTION]
> Usage:

```bash
aicancel
```

> [!IMPORTANT]
> Description:

```ini
Cancels the current operation
```

> [!WARNING]
> Command:

```bash
aifollow
```

> [!CAUTION]
> Usage:

```bash
aifollow
aifollow <player_name>
```

> [!IMPORTANT]
> Description:

```ini
NPCs follow the administrator (if no argument is provided) or the specified player until death
```

> [!WARNING]
> Command:

```bash
aicome
```

> [!CAUTION]
> Usage:

```bash
aicome
```

> [!IMPORTANT]
> Description:

```ini
NPCs move to your position
```

> [!WARNING]
> Command:

```bash
aifleet
```

> [!CAUTION]
> Usage:

```bash
aifleet <fleet_name>
```

> [!IMPORTANT]
> Description:

```ini
Creates an NPC fleet
```

> [!WARNING]
> Command:

```bash
fleetlist
```

> [!CAUTION]
> Usage:

```bash
fleetlist
```

> [!IMPORTANT]
> Description:

```ini
Prints the list of NPC fleets to the console
```

### ADMIN COMMANDS

> ---
>
> [!TIP]
> Команда:

```bash
aicreate
```

> [!CAUTION]
> Применение:

```bash
aicreate <count> <name>
```

> [!IMPORTANT]
> Описание:

```ini
Создаёт NPC
```

> ---
>
> [!TIP]
> Команда:

```bash
aidestroy
```

> [!CAUTION]
> Применение:

```bash
aidestroy
```

> [!IMPORTANT]
> Описание:

```ini
Уничтожает все NPC
```

> ---
>
> [!TIP]
> Команда:

```bash
aicancel
```

> [!CAUTION]
> Применение:

```bash
aicancel
```

> [!IMPORTANT]
> Описание:

```ini
Отменяет операцию
```

> ---
>
> [!TIP]
> Команда:

```bash
aifollow
```

> [!CAUTION]
> Применение:

```bash
aifollow
aifollow <player_name>
```

> [!IMPORTANT]
> Описание:

```ini
NPC следят за администратором (если нет аргумента) или игроком до самой смерти
```

> ---
>


> [!CAUTION]
> Применение:

```bash
aicome
```

> [!IMPORTANT]
> Описание:

```ini
NPC прийдут на вашу позицию
```

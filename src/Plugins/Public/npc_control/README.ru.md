### ADMIN COMMANDS

> [!WARNING]
> Команда:

```bash
aicreate
```

> [!CAUTION]
> Применение:

```bash
aicreate <count> <name>
```

> [!IMPORTANT]
> Описание:

```ini
Создаёт NPC
```

> [!WARNING]
> Команда:

```bash
aidestroy
```

> [!CAUTION]
> Применение:

```bash
aidestroy
```

> [!IMPORTANT]
> Описание:

```ini
Уничтожает все NPC
```

> [!WARNING]
> Команда:

```bash
aicancel
```

> [!CAUTION]
> Применение:

```bash
aicancel
```

> [!IMPORTANT]
> Описание:

```ini
Отменяет операцию
```

> [!WARNING]
> Команда:

```bash
aifollow
```

> [!CAUTION]
> Применение:

```bash
aifollow
aifollow <player_name>
```

> [!IMPORTANT]
> Описание:

```ini
NPC следят за администратором (если нет аргумента) или игроком до самой смерти
```

> [!WARNING]
> Команда:

```bash
aicome
```

> [!CAUTION]
> Применение:

```bash
aicome
```

> [!IMPORTANT]
> Описание:

```ini
NPC прийдут на вашу позицию
```

> [!WARNING]
> Команда:

```bash
aifleet
```

> [!CAUTION]
> Применение:

```bash
aifleet <fleet_name>
```

> [!IMPORTANT]
> Описание:

```ini
NPC флот создаётся
```

> [!WARNING]
> Команда:

```bash
fleetlist
```

> [!CAUTION]
> Применение:

```bash
fleetlist
```

> [!IMPORTANT]
> Описание:

```ini
Выводит в консоль NPC флоты
```

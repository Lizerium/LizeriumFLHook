## ADMIN COMMANDS

> command:

```bash
/getstats
```

> [!CAUTION]
> Применение:

```bash
/getstats

```

> [!IMPORTANT]
> Описание:

```ini
Показывает статус целей в системе в формате charname=%s clientid=%u loss=%u lag=%u pingfluct=%u saturation=%u txqueue=%u
```

> [!WARNING]
> Команда:

```bash
/kick
```

> [!CAUTION]
> Применение:

```bash
/kick <charname>

```

> [!IMPORTANT]
> Описание:

```ini
Выгоняет игрока с сервера
```

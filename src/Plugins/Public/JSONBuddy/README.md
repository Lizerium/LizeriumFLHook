### 🎄 JSON BUDDY 🎄

> [!IMPORTANT]
>
> - Plugin loaded: jsonbuddy **(JSONBuddy.dll)**
>
>   Описание

```ini
Загружает все пропуска (сканеры Tractor) и корабли (Ship),
показывается ваш пинг и пинг цели, позволяет админу кикать игрока и смотреть его статистику подключения
```

> 2.  **..\\data\\equipment\\misc_equip.ini**

## ADMIN COMMANDS

> command:

```bash
/getstats
```

> [!CAUTION]
> Применение:

```bash
/getstats

```

> [!IMPORTANT]
> Описание:

```ini
Показывает статус целей в системе в формате charname=%s clientid=%u loss=%u lag=%u pingfluct=%u saturation=%u txqueue=%u
```

> ---
>
> [!WARNING]
> Команда:

```bash

```

```bash

```

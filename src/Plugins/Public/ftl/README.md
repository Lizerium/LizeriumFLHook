### FTL

> [!IMPORTANT]
>
> - Plugin loaded: ftl **(ftl.dll)**
>
>   Описание

```ini
Добавляет устройства телепорта разных уровней, и специальные ограничения на стыковку связанное с наличием груза
```

> ---

> 1.  **accpath + \\flhookuser.ini**
> 2.  [Пример конфигурации `ftl.ini`](../../../Binaries/bin-conf/flhook_plugins/ftl.ini)
> 3.  [Пример конфигурации `ftlbadsystems.ini`](../../../Binaries/bin-conf/flhook_plugins/ftlbadsystems.ini)

> ---

> [!NOTE]
> Функция:

```bash
UserCmd_FTL
```

> [!WARNING]
> Команда:

```bash
/ftl
```

> [!CAUTION]
> Применение:

```bash
/ftl
/ftl [x] [y] [z]
```

> [!IMPORTANT]
> Описание:

```ini
Телепорт со смещением на указанные координаты к выделенному объекту
Например: /ftl 100 -100 -100 -> телепорт на 100 вправо, 100 вниз, 100 назад от выделенного объекта
```

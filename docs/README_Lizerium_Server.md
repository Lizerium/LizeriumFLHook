### HOOK

⤴️[Оглавление](#Оглавление)

>

```bash
/set diemsg <visibility>
```

```ini
Устанавливает видимость сообщений о смерти. Параметры: all, system, self, none.
```

> ---

```bash
/set diemsgsize <size>
```

```ini
Устанавливает размер текста сообщений о смерти. Параметры: small, default.
```

> ---

```bash
/set chatfont <size> <style>
```

```ini
Установка шрифта сообщений в чате. Варианты small, default или big для <size> и default, bold, italic или underline для <style>
```

> ---

```bash
/ignore <charname> <flags>
```

```ini
Игнорирует все сообщения от данного персонажа.
Возможные флаги: p - Влияет только на приватный чат i - <charname> может совпадать частично
Примеры:
/ignore SomeDude - игнорирует все сообщения в чате от SomeDude
/ignore PlayerX - игнорирует все сообщения в приватном чате от PlayerX
/ignore idiot - игнорирует все сообщения в чате от игроков, чье имя содержит idiot (e.g. [XYZ]IDIOT, MrIdiot, etc)
/ignore Fool - игнорирует все сообщения в приватном чате от игроков, чье имя содержит fool
```

> ---

```bash
/ignoreid <client-id> <flags>
```

```ini
Игнорирует все сообщения от персонажа с ассоциированным идентификатором клиента (см. /id). Используйте флаг p, чтобы повлиять только на приватный чат.
```

> ---

```bash
/ignorelist
```

```ini
Отображает все игнорируемые в данный момент символы.
```

> ---

```bash
/delignore <id> <id2> <id3> ...
```

```ini
Удаляет символы с соответствующим идентификатором игнорирования (см. /ignorelist) из списка игнорирования. * deletes all.
```

> ---

```bash
/ids
```

```ini
Список всех персонажей с соответствующими идентификаторами клиента.
```

> ---

```bash
/id
```

```ini
Предоставляет ваш собственный идентификатор клиента.
```

> ---

```bash
/help <command>
```

```ini
Отображение экрана справки. Указание a <command> дает подробную информацию для этой команды.
```

### AUTOBUY

⤴️[Оглавление](#Оглавление)

>

```bash
/autobuy <param> [<on/off>]

/autobuy info - показать информацию об автокупке
/autobuy missiles on/off - включить/выключить автокупку ракет
/autobuy torps on/off - включить/выключить автокупку торпед
/autobuy mines on/off - включить/выключить автокупку мин
/autobuy cd on/off - включить/выключить автокупку блокираторов
/autobuy cm on/off - включить/выключить автокупку обманок
/autobuy reload on/off - включить/выключить автокупку нанороботов и батарей щита
/autobuy all on/off - включить/выключить автокупку всех расходников

Примеры:
/autobuy info - показать информацию об автокупке
/autobuy all on - включить автокупку всех расходников
/autobuy torps off - отключить автокупку торпед
```

```ini
Управляет автокупкой расходников
```
